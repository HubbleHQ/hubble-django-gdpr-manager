
Django GDPR Manager
=====

Package to handle hubbles GDPR data within Django. 

Quick start
-----------

1. Add "gdpr_manager" to your INSTALLED_APPS setting like this:
    ```
    INSTALLED_APPS = [
        ...,
        "gdpr_manager",
    ]
    ```
